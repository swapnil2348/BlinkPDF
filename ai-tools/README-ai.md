# AI Tools Added

- ai-translator: AI PDF Translator — translate entire PDFs to another language (page-by-page preview).
- ai-rewriter: AI PDF Rewriter — improve tone, grammar, shorten/expand text.
- ai-qa: AI PDF Question Answering — ask questions about the PDF and get answers with page refs.
- ai-form-fill: AI PDF Form Autofill — auto-fill form fields using provided profile data.
- ai-redact: AI Content Detector + Redaction — detect sensitive info and redact it.
- ai-compare: AI PDF Compare — compare two PDFs and highlight differences.
- ai-pptx: AI PDF → PowerPoint Converter (Smart Slides).
- ai-html: AI PDF → HTML / Blog Article Converter.
- ai-advanced-summarizer: AI Summarizer (advanced) — section summaries, bullets, action items.
- ai-ocr-advanced: AI OCR (advanced) — OCR with layout and table extraction.
- ai-metadata: PDF Metadata Extractor (improved).
- ai-extract-tables: AI Table Extractor — extract tables as CSV/Excel.
- ai-image-extract: AI Image Extractor — extract and tag images inside PDF.
- ai-reflow: AI Reflow (make PDF mobile-friendly / reflow text).
- ai-legal-review: AI Contract / Legal Clause Reviewer — explain clauses and flag risky terms.
- ai-financial-extract: AI Financial Data Extractor — pull amounts, dates, totals.
- ai-keyphrase: AI Keyphrase & Topic Extractor — produce keywords, tags, TOC.
- ai-speech: AI PDF → Audio (Text-to-Speech) — generate narrated MP3.
- ai-redline: AI Redline (track changes) — produce redline diff document.
- ai-ocr-to-searchable: Make PDF Searchable — OCR & embed text layer.
- ai-security-scan: Security Scan — find hidden metadata, trackers, embedded scripts.
- ai-smart-fill: Smart Field Suggestion — suggest values for empty fields.
- ai-bulk: Bulk Processor — run any AI action across many PDFs.


### IMPORTANT: Key handling
Do NOT paste keys into chat or commit them to the repository. If you already pasted a key publicly, rotate it immediately.
Set GEMINI_API_KEY as an environment variable or use a secrets manager.
